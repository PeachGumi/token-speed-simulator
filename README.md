# Token Speed Simulator

LLM（大規模言語モデル）の出力速度を体感するためのブラウザベースのシミュレーターです。

## 機能

- 5〜100 tok/s のプリセット速度選択
- カスタム速度指定（1〜500 tok/s）
- テキスト入力またはデフォルトテキストでアニメーション表示
- 実測速度・トークン数・経過時間のリアルタイム表示
- Start / Stop / Clear 操作

## 使い方

1. `index.html` をブラウザで開く
2. 速度を選択する（プリセットボタン or 直接入力）
3. テキストエリアにテキストを貼り付ける（任意）
4. **Start** をクリックしてアニメーション開始

## 技術

- HTML / CSS / JavaScript のみ（外部依存なし）
- `requestAnimationFrame` による高精度なアニメーション

## 注意事項

- 表示される速度は目安です。デバイスの性能やブラウザの負荷によって、実際の表示速度が異なる場合があります
- 長いテキストや低速なデバイスでは、アニメーションがカクつくことがあります
